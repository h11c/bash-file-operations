## Bash File Operations Script

A Bash script that automates basic directory and file operations.

### What it does
- Creates a directory called `bash_demo`
- Navigates into the directory
- Creates a file called `demo.txt`
- Writes text to the file including the current date
- Displays the file contents

### How to run
```bash
chmod +x file_ops.sh
./file_ops.sh
